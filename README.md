# robot-rl-simulation

2D-симуляция робота, который самостоятельно учится ориентироваться в пространстве и достигать цели, избегая препятствий — без заранее прописанных правил.

## Как это работает

Робот находится на карте с препятствиями и целью. Он не видит карту целиком — только показания 5 сенсорных лучей вокруг себя (аналог лидара), расстояние до цели и угол до цели. На основе этих 7 чисел нейросеть принимает решение: повернуть влево, повернуть вправо или ехать вперёд.

Обучение происходит через алгоритм PPO (Proximal Policy Optimization) — робот делает тысячи попыток, получает награду за достижение цели и штраф за столкновения, и постепенно выучивает оптимальную стратегию.

**Пайплайн:**
- Среда написана с нуля на Pygame + Gymnasium
- 5 сенсорных лучей (raycasting) как входные данные агента
- Обучение через PPO (Stable-Baselines3)
- Финальная средняя награда: 98.5 за эпизод
- Агент достигает цели в среднем за 15 шагов

## Технологии

- Python
- Pygame
- Gymnasium
- Stable-Baselines3
- PyTorch

## Как запустить

Установи зависимости:
pip install -r requirements.txt

Обучи агента:
python train.py

Запусти визуализацию:
python visualize.py

## Структура проекта

- `robot_env.py` — среда (физика, сенсоры, награды)
- `train.py` — обучение агента
- `visualize.py` — визуализация обученного агента
- `play.py` — тест модели в консоли
- `models/` — сохранённые модели

## Результаты

Агент обучился за 300 000 шагов и стабильно достигает цели, огибая препятствия. Средняя награда за эпизод — 98.5, что соответствует достижению цели почти в каждом эпизоде.
